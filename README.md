
# Nneka Ifediorah – Professional Portfolio

This is a professional portfolio website built with [Next.js](https://nextjs.org) and TypeScript, featuring:

- Home/About Me
- Projects
- Blog
- Testimonials
- Skills
- Contact
- Downloadable CV ([public/cv/Nneka Ifediorah - Software Engineer.pdf](public/cv/Nneka%20Ifediorah%20-%20Software%20Engineer.pdf))

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Customization

Edit the content in the `/src/app` directory to update sections. Replace placeholder text and add your own projects, blog posts, testimonials, and skills.
